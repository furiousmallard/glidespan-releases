# Glidespan Releases

This repository is used for hosting public releases of Glidespan, as well as the changelog.

Glidespan is **not** open source and this repo *DOES NOT* contain any source code for Glidespan.

If you have a feature request or bug report, please [file an issue](https://github.com/furiousmallard/glidespan-releases/issues/new/choose).

## Download

**[Download Glidespan](https://glidespan.app/download)** - Always get the latest version from our website.

## Requirements

- macOS 14 Sonoma or later
- Apple Silicon or Intel Mac

Follow us on [Bluesky](https://bsky.app/profile/glidespan.app) for updates and announcements.
